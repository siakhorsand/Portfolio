AI MathPad

A program that utilizes Convolutional neural networks (CNNs) and image processing techniques to recognize handwritten digits and mathematical operations drawn on a digital canvas. The system leverages computer vision and deep learning to preprocess and classify user input in real time, and ultimately returning the correct answer to the mathematical operation. Learn More

Comparative Analysis of Machine Learning Model Performance Across Three Datasets: A Technical Evaluation.

Conducted a comparative analysis of supervised machine learning classifiers—Random Forest, Support Vector Machine (SVM), and Logistic Regression—across multiple binary-class datasets. The project involved hyperparameter tuning, cross-validation, and performance evaluation using Python's scikit-learn. Demonstrated methods in model selection, data preprocessing, and visualization while working with real-world datasets from the UCI Machine Learning Repository. Source code can be provided upon request.

Dynamic Valet Business Management App

In the final development stages of an iOS app, currently being beta-tested for my part-time valet job, to dynamically manage and track shifts and cars with a responsive grid layout to boost efficiency by 25%. Employed MVVM architecture for data persistence, along with Firebase to handle the backend and user authentication. The app is designed to be user-friendly and intuitive, with a focus on real-time updates and notifications.
